# Invitación de San Valentín

Proyecto estático con un mensaje bonito y didáctico para invitar a tu pareja a celebrar el 14 de febrero.

## Cómo verlo

1. Abre el archivo index.html en tu navegador.
2. Presiona el botón para revelar el mensaje especial.

## Personalización rápida

- Cambia el texto principal en index.html.
- Ajusta colores y animaciones en styles.css.
- Modifica los corazones flotantes en script.js.
